# absolute-convergent-series-can-be-reordered

## 主結果
$\sum a_n, \quad a_n \in \mathbb{C}$ が絶対収束するなら

任意の置換 $\sigma: \mathbb{N} \rightarrow \mathbb{N}$ について

$\sum a_{\sigma(n)}$ は絶対収束し、 $\sum a_n=\sum_{\sigma(n)}$ となる
